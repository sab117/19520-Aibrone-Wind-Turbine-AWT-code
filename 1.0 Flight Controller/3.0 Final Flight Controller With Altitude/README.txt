Final Flight Controller 
By Saboor Rashid and Qamar Ilyas
Last modified: 26/02/2023

This code is the final code that is used as the flight controller for the AWT. It uses a PCA9548A MUX switcht to switch bewteen MPU-6050 and the BMP-180. The BMP-180 allows
the altitude to be determined as well as the temperature of the external environment.
